("금액", "개수", "건수")│  ├─ YES → col.number()│  └─ NO → 다음 단계│├─ 상태 필드인가? ("상태", 색상 표시)│  ├─ YES → col.statusText()│  └─ NO → 다음 단계│├─ 토글 스위치인가? (Toggle 아이콘)│  ├─ YES → col.switch()│  └─ NO → 다음 단계│├─ 버튼인가? ("수정", "삭제" 등)│  ├─ YES → col.button() 또는 col.modalOpen()│  └─ NO → 다음 단계│└─ 기본 텍스트   ├─ 긴 텍스트? → col.tooltip()   └─ 짧은 텍스트 → col.text()`---## 실전 예시: AI 코드 생성 과정### 입력: 기획서 분석`기획서 142p: 타거래소 이동 지원 신청 목록테이블 컬럼:1. 접수일시 (2026-01-15 14:30:00)2. MID (A12345678A) - underline, 클릭 가능3. UID (U12345678) - underline, 클릭 가능4. 회원명 (홍길동) - underline, 클릭 가능5. 검증 일시 (2026-01-16 10:00:00)6. 관리자 (admin001)7. 상태 (접수, 승인, 반려, 입금대기, 대면 인증 대기)8. 월 평균 거래 금액 (1,234,567원)`### 출력: AI 생성 코드``tsximport React, { useMemo } from 'react';import { GridColDef } from '@bos/components';import { useNavigate } from 'react-router';import { useColumnBuilder } from 'common/components/TableCells';interface ICompetitorMoveSupportData {    id: string;    applyDate: string;    mid: string;    uid: string;    memberName: string;    verifyDate: string | null;    adminId: string | null;    status: 'APPLY' | 'APPROVE' | 'REJECT' | 'DEPOSIT_WAIT' | 'OFFLINE_AUTH_WAIT';    avgTradeAmount: number;}const STATUS_TEXT_MAP: Record<string, string> = {    APPLY: '접수',    APPROVE: '승인',    REJECT: '반려',    DEPOSIT_WAIT: '입금대기',    OFFLINE_AUTH_WAIT: '대면 인증 대기'};const CompetitorMoveSupportList = () => {    const navigate = useNavigate();    const col = useColumnBuilder<ICompetitorMoveSupportData>();    const columns = useMemo<GridColDef<ICompetitorMoveSupportData>[]>(       () => [          // 1. 접수일시 → 날짜 컬럼          col.date('applyDate', '접수일시'),          // 2. MID → 링크 컬럼 (underline, 클릭 가능)          col.link('mid', 'MID', {             href: (row) => `detail/${row.id}`          }),          // 3.
