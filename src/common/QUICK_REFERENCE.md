# Column Builder DSL - Quick Reference## 1분 안에 시작하기### Import`tsximport { useColumnBuilder } from 'common/components/TableCells';import { GridColDef } from '@bos/components';`### 기본 사용``tsxconst col = useColumnBuilder<IYourDataType>();const columns = useMemo<GridColDef<IYourDataType>[]>(() => [  col.text('name', '이름'),  col.link('mid', 'MID', { href: (row) => `/detail/${row.id}` }),  col.statusText('status', '상태', { textMap: STATUS_MAP }),  col.number('amount', '금액', { suffix: '원' })], [col]);``---## API Cheat Sheet### 📝 기본 텍스트`tsxcol.text('field', 'Label')col.text('field', 'Label', { flex: 1.5, minWidth: 200, fallback: '없음' })`### 📅 날짜`tsxcol.date('createdAt', '생성일시')col.date('modifiedAt', '수정일시', { format: 'YYYY-MM-DD' })`### 🔢 숫자`tsxcol.number('amount', '금액', { suffix: '원' })col.number('count', '건수', { suffix: '건' })`### 🔢 숫자 + 단위 (줄바꿈)`tsx// 숫자와 단위를 줄바꿈하여 표시 (단위는 회색)col.numberWithUnit('incomeAmount', '소득금액', { unit: 'KRW' })col.numberWithUnit('balance', '잔액', { unit: 'BTC', decimals: 8 })`### 🔗 링크``tsx// href로 네비게이션col.link('mid', 'MID', { href: (row) => `/detail/${row.id}` })// onClick으로 커스텀 핸들러col.link('name', '이름', { onClick: (row, e) => handleClick(row) })// 외부 링크col.link('url', 'URL', { href: (row) => row.url, external: true })``### 🎨 상태 텍스트`tsxcol.statusText('status', '상태', {  textMap: { 'ACTIVE': '활성', 'INACTIVE': '비활성' },  colorMap: { 'ACTIVE': 'success.main', 'INACTIVE': 'error.main' },  bold: true})`### 🔘 버튼`tsxcol.button('id', '액션', {  label: '수정',  onClick: (row) => handleEdit(row),  color: 'primary'})`### 🎚️ 스위치`tsxcol.switch('isActive', '활성', {  onChange: (row, checked) => handleToggle(row, checked),  showLabel: true,  labels: { on: '활성', off: '비활성' }})`### 💬 Tooltip`tsxcol.tooltip('description', '설명', { maxLength: 20 })`### 🏷️ Chip`tsxcol.chip('tag', '태그', { color: 'primary', size: 'small' })`### 🔔 Badge`tsxcol.badge('count', '알림', { color: 'error', max: 99 })`### 📋 Modal Open`tsxcol.modalOpen('id', '상세', {  label: '상세보기',  onOpen: (row) => handleOpen(row)})`### ✏️ Input`tsxcol.input('name', '이름', {  onChange: (row, value) => handleUpdate(row, value),  placeholder: '이름 입력'})`### 📑 Select`tsxcol.select('status', '상태', {  options: [    { value: 'ACTIVE', text: '활성' },    { value: 'INACTIVE', text: '비활성' }  ],  onChange: (row, value) => handleChange(row, value)})`### 📄 Multi Line`tsxcol.multiLine('tags', '태그', { maxLines: 3, gap: 0.5 })`---## Column Options모든 메서드는 마지막 파라미터로 공통 옵션을 받습니다:`tsx{  flex?: number           // 기본: 1  minWidth?: number       // 기본: 100  width?: number          // ⚠️ 사용 지양 (flex 권장)  align?: 'left' | 'center' | 'right'  headerAlign?: 'left' | 'center' | 'right'}`**예시**:`tsxcol.text('name', '이름', { flex: 1.5, minWidth: 200, align: 'left' })`---## 컬럼 타입 선택 가이드| 기획서 키워드 | 사용할 메서드 | 예시 ||--------------|--------------|------|| "일시", "날짜", "Date" | `col.date()` | 접수일시, 수정일시 || MID, UID, 회원명 (underline) | `col.link()` | MID, UID, 이름 || "금액", "개수", "건수" | `col.number()` | 금액, 평균 거래액 || "상태", "Status" | `col.statusText()` | 상태, 승인여부 || "활성", "사용여부" (Toggle) | `col.switch()` | 활성화, 사용여부 || "수정", "삭제" (버튼) | `col.button()` | 수정, 삭제 || "상세보기", "조회" (버튼) | `col.modalOpen()` | 상세보기, 이력조회 || "설명", "비고" (긴 텍스트) | `col.tooltip()` | 설명, 내용 || 태그, 라벨 | `col.chip()` | 태그, 카테고리 || 알림 개수 | `col.badge()` | 알림, 메시지 || 기본 텍스트 | `col.text()` | 관리자, 이름 |---## 금지 사항 ❌### 1. renderCell 직접 사용 금지`tsx// ❌ 금지{  field: 'mid',  renderCell: ({ value }) => <Box>{value}</Box>}// ✅ 권장col.text('mid', 'MID')`### 2. width 사용 금지`tsx// ❌ 금지col.text('name', '이름', { width: 120 })// ✅ 권장col.text('name', '이름', { flex: 0.8, minWidth: 120 })`### 3. 상태에 Badge 사용 금지`tsx// ❌ 금지col.chip('status', '상태')col.badge('status', '상태')// ✅ 권장col.statusText('status', '상태', { textMap: STATUS_MAP })`---## 실전 패턴### 패턴 1: 기본 목록 페이지``tsxconst col = useColumnBuilder<IListData>();const columns = useMemo<GridColDef<IListData>[]>(() => [  col.date('createdAt', '등록일시'),  col.link('mid', 'MID', { href: (row) => `/detail/${row.id}` }),  col.text('name', '이름'),  col.statusText('status', '상태', { textMap: STATUS_MAP }),  col.button('id', '액션', {    label: '수정',    onClick: (row) => handleEdit(row)  })], [col]);``### 패턴 2: 통계 포함 목록`tsxconst columns = useMemo<GridColDef<IData>[]>(() => [  col.text('category', '카테고리'),  col.number('totalAmount', '합계', { suffix: '원' }),  col.number('count', '건수', { suffix: '건' }),  col.number('average', '평균', { suffix: '원' })], [col]);`### 패턴 3: 인라인 편집 목록`tsxconst columns = useMemo<GridColDef<IData>[]>(() => [  col.text('name', '이름'),  col.input('memo', '메모', {    onChange: (row, value) => handleUpdateMemo(row.id, value)  }),  col.select('status', '상태', {    options: STATUS_OPTIONS,    onChange: (row, value) => handleUpdateStatus(row.id, value)  }),  col.switch('isActive', '활성', {    onChange: (row, checked) => handleToggle(row.id, checked)  })], [col]);`---## 자주 묻는 질문**Q. 여러 컬럼에 같은 href를 사용하려면?**``tsxconst getDetailHref = (row: IData) => `/detail/${row.id}`;const columns = [  col.link('mid', 'MID', { href: getDetailHref }),  col.link('uid', 'UID', { href: getDetailHref }),  col.link('name', '이름', { href: getDetailHref })];``**Q. 조건부 스타일 적용하려면?**`tsxcol.statusText('status', '상태', {  textMap: STATUS_MAP,  colorMap: {    'ACTIVE': 'success.main',    'PENDING': 'warning.main',    'INACTIVE': 'error.main'  }})`**Q. 버튼을 조건부로 비활성화하려면?**`tsxcol.button('id', '삭제', {  label: '삭제',  onClick: (row) => handleDelete(row),  disabled: (row) => row.status === 'LOCKED'})`**Q. Cell 컴포넌트를 직접 사용해도 되나요?**가능하지만 Column Builder DSL 사용을 권장합니다. 매우 특수한 케이스에만 직접 사용하세요.``tsximport { LinkCell } from 'common/components/TableCells';{  field: 'mid',  renderCell: ({ value, row }) => (    <LinkCell value={value} href={`/detail/${row.id}`} />  )}``---## 더 알아보기- [README.md](./README.md) - 상세 문서- [REFACTORING_EXAMPLE.md](./REFACTORING_EXAMPLE.md) - Before/After 비교- [AI_AUTOMATION_STRATEGY.md](./AI_AUTOMATION_STRATEGY.md) - AI 자동화 가이드---**마지막 업데이트**: 2026-02-16
